Ontology Specification Draft

Introduction back to ToC

This document contains the first version of the agrixel ontology, a standard for the semantic representation of agricultural data at a 1 meter x 1 meter scale. It is designed to facilitate the integration, interoperability, and exploitation of heterogeneous agricultural data using artificial intelligence and computer vision techniques, while promoting the adoption of FAIR principles in data sharing and standardization.
